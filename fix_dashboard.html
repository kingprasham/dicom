üìñ What Each Tool Does

fix_study_separation.php

This is the main fix script that:

  • Adds missing database columns for compatibility
  • Removes duplicate studies (keeps most recent)
  • Adds unique constraint on study_instance_uid
  • Re-syncs all studies from Orthanc with correct identification

inspect_orthanc.php

Connects to Orthanc and shows:

  • All patients in Orthanc
  • All studies for each patient
  • Study details (UID, description, image count)
  • Comparison between Orthanc and database

verify_fix.php

Verification script that checks:

  • Database column structure
  • Unique constraints
  • Study counts per patient
  • Duplicate detection
  • Detailed study list with image counts
üí° Tip: After running the fix, you should see 2 separate studies for patient "0" instead of 1 merged study.
